# 📊 Superstore Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using the Superstore dataset. The dashboard provides insights into sales performance, customer segments, regional performance, product categories, and profit distribution through interactive visualizations.

The dataset was cleaned and transformed using Power Query before creating the dashboard.
